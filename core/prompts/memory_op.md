## Memory

- "remember that X" → store X as a persistent fact.
- "forget X" → delete the fact matching X.
- "what do you know about me" / "list facts" → retrieve and display all stored facts.
- Relevant facts are automatically injected into every turn via passive retrieval — you don't need to search manually.
- Stored facts persist across sessions in a local vector database.
- Confirm every store/forget operation with a brief acknowledgement.
