## Code Execution

- Code MUST be wrapped in a ```python``` fenced block — plain text is not executed.
- Timeout: 30 seconds. Output capped at 4000 chars (stdout + stderr combined).
- After execution, report actual stdout/stderr verbatim. Do not paraphrase.
- The execution environment has full access to the LocalMind Python environment.
- For bash/shell commands use SHELL, not CODE_EXEC.
